About Project

ARTVIL이라는 가구전문점의 사이트를 제작해보았습니다.
기업소개사이트의 특성에 맞게 심플하게 구성하고 제품과 주요정보만을 소개하고
큰 효과 대신 각 버튼이나 요소에 마우스를 올리면 css값이 변경되는 간단한 효과만 주었습니다.

Design

가구전문 사이트이기때문에 한 페이지에 주요제품과 매장정보를 보여주는 게 중요하다 생각해 메인 상단에는 크게 대표가구이미지를 넣고 자동으로 롤링되는 배너를 만들고 페이지 내에는 제품 이미지들을 한데 모아 보여주고 지도를 넣어 매장 위치를 찾기 쉽도록 하였습니다.

Publishing

CODING

메인 배너에 prependTo와 setInterval 값을 이용하여 자동으로 이미지들이 슬라이딩 되고 양측에 버튼을 누르면 이전과 다음 이미지로 넘어갈 수 있게 하였습니다.
갤러리에 마우스를 올리면 텍스트가 뜨고 transform scale로 이미지가 살짝 커지며 배경색이 불투명하게 바뀌는 효과를 주었습니다.
하단에 위치한 탑버튼을 누르면 화면 윗쪽으로 가 불필요하게 위로 스크롤할 일이 없게 하였습니다.

DEVELOP

상단 네비게이션 메뉴에 마우스를 올리면 각각 원래의 li값보다 넒이가 큰 메뉴가 transition기능이 적용되서 내려왔지만 서브페이지를 안만들었기 때문에 빼었습니다.
