About Project

최근 SNS에서 핫한 홍대의 젤라또 전문점이지만 사이트가 없어 제작해보았습니다.
실제 제품 사진을 사용하여 생생한 젤라또의 느낌을 전달하려 하였고 젤라또의 달콤한 느낌을
살리려 파스텔톤을 사용했지만 진한 팝아트적인 색상으로 포인트를 주어 파스텔톤이 갖고있는
여리여리한 느낌을 없애고 조금 더 임팩트 있고 젊은 세대층의 눈길을 끌 수있게 만들었습니다.

Design

MAIN PAGE

풀사이즈 화면에 네온효과의 로고와 튀는 색상의 메뉴 네비게이션이 보여 한눈에 보이도록 하였고, 보색 관계의 듀오톤을 써 화면 속 젤라또 이 미지를 강조하며 젤라또 전문점 페이지임을 알 수 있게 하였습니다. 풀사이즈 화면이기 때문에 스크롤을 한번 내리면 바로 밑 소개페이지로 갈 수 있도록 만들어 보았습니다.

Publishing

CODING

메인화면에서 스크롤을 내리면 다음페이지로 가게하는 스크롤값을 지정해
만들었습니다. 메인 인스타그램 네비에 id값을 링크해 페이지 내 인스타그
램 영역으로 갈 수 있게 하였습니다. 메뉴에는 text-shadow를 써 그림자효
과로 포인트를 주었습니다. 버튼이나 이미지에는 hover 효과를 줘 배경색이
바뀌는 효과를 주었습니다. 하단에 검색창에는 placeholder값을 주어 텍스
트를 입력할 수 있도록 하였습니다.

DEVELOP

플러그인 적용에 대해 잘 알지 못하고 만든 페이지여서 스크롤을 내리면 적
용되는 애니메이션 효과를 잘 만들지 못하였습니다.
또한 인스타그램 사진들이 자동으로 바뀌는 기능과 사진을 누르면 라이트
박스창이 뜨는 기능을 같이 주고 싶었으나 위치값을 안 지정하고 만들어 라
이트박스를 적용하기 어려워 다시 시도해 보려고 합니다.
